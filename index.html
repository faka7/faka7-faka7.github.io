CosmoBot

Chatbot Educativo sobre Exploración Espacial

Un proyecto del equipo "Choclo con Queso"

¿Qué es CosmoBot?

CosmoBot es un chatbot educativo diseñado para enseñar sobre exploración espacial y cosmología. Utiliza un modelo de lenguaje grande (LLM) para ofrecer respuestas naturales y personalizadas, buscando fomentar la educación accesible e inspirar curiosidad sobre la ciencia.

Tecnologías Clave

El proyecto se construyó sobre un stack moderno de IA y desarrollo web:

  • Modelo LLM

    Hugging FaceH4/mistral-7b-grok (arquitectura Mistral-7B con estilo "Grok").

  • Backend

    Python 3.11 con el micro-framework Flask para servir la API y el modelo.

  • Motor de IA

    Bibliotecas `torch` (para CUDA) y `transformers` de Hugging Face.

  • Frontend

    HTML simple para la interfaz del chat (index.html) y el panel de métricas (metrics.html).

Hitos por Sprint

Sprint 1: Planificación y Fundación

  • Investigación de modelos (Grok-2/4).
  • Creación del Acta de Constitución.
  • Definición del Product Backlog inicial.
  • Registro de Interesados.

Sprint 2: Desarrollo del Prototipo

  • Configuración del entorno Python y Mistral-7B-Grok.
  • Pruebas de prompts básicos (Marte, estrellas).
  • Diseño de la interfaz de chat (index.html).
  • Desarrollo del sistema de respuestas (Backend Flask).

Sprint 3: Métricas y Validación

  • Diseño del panel de métricas (metrics.html).
  • Implementación del endpoint /metrics.
  • Validación de privacidad (no almacenar PII).
  • Verificación con checklist ético.
