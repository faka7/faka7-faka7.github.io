Análisis Predictivo con Machine Learning

Proceso completo de implementación de modelos predictivos en tres conjuntos de datos diferentes

Introducción

Este proyecto implementa modelos de machine learning para resolver problemas de regresión y clasificación en tres conjuntos de datos diferentes. A continuación se presenta el proceso completo desde la selección de datos hasta las conclusiones.

1 Selección de Datos

Medical Cost

Datos de seguros médicos para predecir costos de atención.

  • 1,338 registros
  • 7 variables
  • Variables numéricas: edad, BMI, hijos
  • Variables categóricas: sexo, fumador, región
Wine Quality

Datos de propiedades de vinos para predecir calidad.

  • 6,497 registros
  • 13 variables
  • Incluye acidez, azúcar, sulfatos, alcohol
  • Calidad como variable objetivo (0-10)
Pima Diabetes

Datos médicos para predecir diabetes.

  • 768 registros
  • 9 variables
  • Variables: glucosa, presión arterial, BMI, etc.
  • Clasificación binaria (diabetes sí/no)

2 Preprocesamiento de Datos

Procesamiento para Regresión
Medical Cost y Wine Quality:
  • División de datos (80% entrenamiento, 20% prueba)
  • Escalado de variables numéricas (StandardScaler)
  • Codificación de variables categóricas (OneHotEncoder)
  • Manejo de valores faltantes (SimpleImputer)
Procesamiento para Clasificación
Pima Diabetes:
  • Manejo de valores cero en variables médicas
  • Imputación de valores faltantes con medianas
  • Escalado de todas las variables
  • División estratificada para mantener proporción de clases

3 Implementación de Modelos

Medical Cost
Regresión Lineal Múltiple

Pipeline con preprocesamiento y modelo lineal

  • Variables numéricas escaladas
  • Variables categóricas codificadas
  • Regresión lineal sin transformación polinomial
Wine Quality
Regresión Polinomial

Grados 2 y 3 probados

  • Preprocesamiento con imputación y escalado
  • Transformación polinomial de características
  • Regresión lineal sobre características polinomiales
  • Evaluación de overfitting
Pima Diabetes
Regresión Logística

Clasificación binaria

  • Preprocesamiento específico para variables médicas
  • Escalado de todas las variables
  • Modelo de regresión logística
  • Análisis de curva ROC y sigmoideal

5 Conclusiones

Resultados por Dataset
  • Medical Cost: La regresión lineal múltiple demostró ser efectiva para predecir costos médicos (R² = 0.784)
  • Wine Quality: La regresión polinomial no mostró buenos resultados, especialmente con grado 3 (R² = 0.034)
  • Pima Diabetes: La regresión logística obtuvo buen desempeño (AUC = 0.772) para clasificación
Lecciones Aprendidas
  • El preprocesamiento adecuado es crucial para el rendimiento del modelo
  • No siempre modelos más complejos (polinomiales) son mejores
  • La evaluación con múltiples métricas proporciona una visión completa
  • El tratamiento de valores missing y outliers afecta significativamente los resultados
